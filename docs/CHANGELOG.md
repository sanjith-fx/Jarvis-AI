# [v1.0] - Initial Stable Release
Release Date: 02-07-2026

## 🎉 Overview

The first stable release of Jarvis AI.

This version introduces an offline AI-powered voice assistant capable of
understanding voice commands, responding with speech, and performing
basic browser automation.

---

## 📌 Known Limitations

- YouTube search does not automatically play videos.
- No wake word support.
- No desktop automation.
- No long-term memory.
- Command system still uses if-else routing.
