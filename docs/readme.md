# 🤖 Jarvis AI

An offline AI-powered personal assistant inspired by Iron Man's JARVIS.

Jarvis combines voice recognition, local AI models, browser automation, and modular architecture to create a scalable personal assistant.

---

## Features

- Voice Recognition
- Offline AI (Ollama + Llama3)
- Voice Responses
- Browser Automation
- YouTube Search
- Time Commands
- Modular Architecture

---

## Tech Stack

- Python
- Ollama
- Llama3
- SpeechRecognition
- PyAudio
- pyttsx3
- Selenium
- ChromeDriver
